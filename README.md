# thesis-case

政治学毕业论文：变通的韧性——基层安全监管部门的"合规-降险"动机

> 导师：凌争 | 方法：案例研究 / 田野调查 / 访谈 / 面板数据分析

---

## 研究主题

在强问责背景下，安全监管部门的街头执行行为遵循怎样的规律？

### 核心概念
- **「变通的韧性」**：变通行为不一定由谋利驱动，在特定条件下凸显
- **合规动机 vs 降险动机**：避责的最优策略不是"留痕"而是"消险"
- **避责型亲商**（留待期刊发表突出，不在毕业设计中突出）
